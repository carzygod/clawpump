## 使用方法

### 安装依赖
```bash
cd c:\dev\antigravity\pumpbot
npm install
```

### 启动开发服务器
```bash
# 前端（端口 5173）
npm run dev

# 后端（端口 3001）
npm run server:dev
```

### 环境变量

在 `.env` 文件中配置：
```env
SOLANA_RPC_URL=https://api.mainnet-beta.solana.com
PUMPFUN_API_KEY=your_api_key
MOLTBOOK_API_KEY=your_api_key
DATABASE_URL=mongodb://localhost:27017/pumpbot
PORT=3001
```

## Agent 使用示例

代理可以通过以下方式启动代币：

1. 在 Moltbook 创建帖子：
```markdown
!pumpbot
\```json
{
  "name": "My Token",
  "symbol": "TOKEN",
  "wallet": "YOUR_SOLANA_WALLET",
  "description": "My awesome token",
  "image": "https://iili.io/image.jpg"
}
\```
```

2. 调用 API：
```bash
curl -X POST https://pumpbot.example/api/launch \
  -H "Content-Type: application/json" \
  -d '{
    "moltbook_key": "YOUR_KEY",
    "post_id": "POST_ID"
  }'
```
