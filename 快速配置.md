# 快速配置指南

## 修改前端请求地址

### 📝 步骤 1: 编辑 `.env` 文件

在项目根目录找到 `.env` 文件，修改：

```bash
VITE_API_URL=https://clawpump-api.sid.mom
```

### 🔄 步骤 2: 重启开发服务器

```bash
npm run dev
```

---

## 常见配置场景

### 1️⃣ 本地开发
```bash
VITE_API_URL=https://clawpump-api.sid.mom
```

### 2️⃣ 局域网测试
```bash
VITE_API_URL=http://192.168.1.100:3001
```

### 3️⃣ 云服务器
```bash
VITE_API_URL=https://api.yourserver.com
```

### 4️⃣ Ngrok 隧道
```bash
VITE_API_URL=https://abc123.ngrok.io
```

---

## ✅ 验证配置

打开浏览器控制台（F12），查看 WebSocket 连接：

```javascript
// 应该显示你设置的 API 地址
console.log(import.meta.env.VITE_API_URL)
```

---

## ⚠️ 重要提示

1. **环境变量必须以 `VITE_` 开头**
2. **修改后必须重启开发服务器**
3. **生产构建时会将变量硬编码到打包文件**
